# Project-3-Circadian-Disruption-in-Medical-Training-Analyzing-Wearable-Data-During-Night-Float
Abstract
Medical training involves rigorous schedules, including overnight shifts that severely disrupt circadian rhythms. This project analyzes simulated wearable health data (representing Apple Watch/Oura Ring metrics) over a one-month surgical rotation. By tracking Sleep Duration and Heart Rate Variability (HRV), we quantify the acute autonomic stress induced by a "Night Float" week and observe the physiological recovery trajectory.
